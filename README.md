# Sales Analysis Project

## Overview

This project analyzes over 10,000 sales records to uncover actionable business insights.  
It combines **SQL** for data extraction and **Excel** for interactive visualization.

## Tools Used

- SQL (JOINs, GROUP BY, CTEs, Subqueries)
- Microsoft Excel (Dashboard & Visualization)
- GitHub (Version Control & Portfolio)

## Steps

1. Collected and cleaned raw sales data (`sales_raw.csv`).
2. Applied SQL queries to identify trends and top-performing products.
3. Built an interactive Excel dashboard (`Sales_Analysis_Project_WahidNageh.xlsx`).

## Key Insights

- Top-performing products contributed **35% of total revenue**.
- Monthly revenue trends visualized for executive reporting.

## Results

- Enabled targeted sales strategies.
- Delivered clear, visual insights for decision-makers.

## Files in this Repository

- `sales_raw.csv` → Raw sales dataset.
- `Sales_Analysis_Project_WahidNageh.xlsx` → Excel dashboard with analysis.
- `SQL_Scripts/sales_analysis_queries.sql` → SQL queries used for data analysis.
- `README.md` → Project documentation.
